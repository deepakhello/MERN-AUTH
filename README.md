# Project Overview

This MERN authentication application demonstrates a complete web development workflow using both frontend and backend technologies.

## What the project does

The application provides a full authentication system where users can:

- Sign up with email and password
- Log in using JWT-based authentication
- Have passwords securely hashed before storage
- Interact with a REST API from a modern frontend
- Use email-related functionality through a configured email service

The backend is implemented with Express and MongoDB, while the frontend uses React, TypeScript, and Vite.

## Key features

- User registration and login
- Secure password hashing with `bcrypt`
- JWT token generation and validation for protected routes
- MongoDB data storage using Mongoose models
- CORS and JSON middleware for API requests
- Production-ready static serving of the frontend build

## Skills and technologies used

### Frontend

- React
- TypeScript

### Backend

- Node.js
- Express
- MongoDB

### Security and authentication

- Password hashing using `bcrypt`
- JSON Web Tokens (`jsonwebtoken`) for authentication
- Secure storage of sensitive configuration in `.env`

### Development practices

- Git version control
- GitHub remote repository

- Post-install build automation for client assets

## Why this project is valuable

This app is a strong example of a modern full-stack web application. It combines frontend UI development with backend API design, database integration, and authentication. The project demonstrates important real-world skills for building secure, maintainable, and deployable web applications.
